Java子串比较

首页 / 新闻资讯 / 正文

以下是比较Java中的字符串和字符串部分的示例举例说明。

示例

public class StringDemo {    public static void main(String[] args) {       String str1 = "";       String str2 = str1.substring(10);       int result = str1.compareTo(str2);       //打印比较的返回值       if (result < 0) {          System.out.println("str1 is greater than str2");       }else if (result == 0) {          System.out.println("str1 is equal to str2");       }else {          System.out.println("str1 is less than str2");       }    } }
